How It Works

Four steps from raw text to verified prediction

01
🧹

Text Cleaning

URLs, HTML tags, punctuation, and stop-words are removed. Text is lowercased and normalised for consistent analysis.

02
📊

TF-IDF Vectorisation

The cleaned text is converted into a 50,000-dimensional numerical feature vector using Term Frequency–Inverse Document Frequency with bigrams.

03
🤖

ML Classification

A trained Logistic Regression classifier — selected from four models — analyses the feature vector and predicts FAKE or REAL with a confidence score.

04
📈

Result & Confidence

The prediction, class probabilities, and confidence percentage are returned instantly and displayed as an interactive visualisation.

About This Project

FakeGuard is an open-source fake news detection system built with Python and scikit-learn. It was trained on the Fake and Real News Dataset (Kaggle) containing over 44,000 labelled news articles.

The system uses TF-IDF feature extraction combined with Logistic Regression to achieve 98%+ accuracy. The backend is served as a serverless Netlify Function, making it free to host and infinitely scalable.
